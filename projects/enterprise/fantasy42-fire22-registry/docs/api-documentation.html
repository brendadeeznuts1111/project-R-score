üîê Authentication

JWT Token Authentication

Most API endpoints require authentication using JWT tokens. Include the token in the Authorization header:

Authorization: Bearer <your-jwt-token>

API Key Authentication

For service-to-service communication, use API keys:

X-API-Key: <your-api-key>

Rate Limiting

  • Authenticated requests: 1000 requests per minute
  • Unauthenticated requests: 100 requests per minute
  • Analytics endpoints: 500 requests per minute

üí° Code Examples

JavaScript/Fetch API

// Get dashboard metrics
const response = await fetch('/api/dashboard/metrics', {
  headers: {
    'Authorization': 'Bearer your-jwt-token',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log('Dashboard metrics:', data);

cURL Examples

# Get dashboard metrics
curl -X GET "http://localhost:3001/api/dashboard/metrics" \
  -H "Authorization: Bearer your-jwt-token" \
  -H "Content-Type: application/json"

# Create new account
curl -X POST "http://localhost:3001/api/domain/operations/create-account" \
  -H "Authorization: Bearer your-jwt-token" \
  -H "Content-Type: application/json" \
  -d '{}'

Python/Requests

import requests

# Get dashboard metrics
headers = {
    'Authorization': 'Bearer your-jwt-token',
    'Content-Type': 'application/json'
}

response = requests.get('http://localhost:3001/api/dashboard/metrics', headers=headers)
data = response.json()
print('Dashboard metrics:', data)
