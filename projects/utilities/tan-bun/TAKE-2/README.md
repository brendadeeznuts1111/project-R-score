    ├── .output/                 # Build output

    ├── public/                  # Static assets

    └── ...                     # Other project files

## Getting Started

### Development
```bash
cd my-tanstack-app
bun run dev
```

### Build & Production
```bash
cd my-tanstack-app
bun run build
bun run start
```

## Features

- ✅ TanStack Start with React Router
- ✅ Tailwind CSS for styling  
- ✅ TypeScript support
- ✅ Bun runtime for fast development
- ✅ Production-ready build system

## Scripts

- `dev` - Start development server
- `build` - Build for production
- `start` - Start production server
- `preview` - Preview production build
