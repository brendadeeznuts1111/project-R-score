The system calculates ROI based on:
- **Component Costs**: Phone + Identity + Warm-up + Device + SIM
- **Success Rate**: Weighted average with synergy bonuses
- **Account Value**: Expected revenue per successful Apple ID
- **ROI Formula**: `(Value √ó Success Rate - Total Cost) √∑ Total Cost √ó 100%`

### Weighted Algorithm
```typescript
const weights = {
```
