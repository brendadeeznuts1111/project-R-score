## üéØ Integration Recommendations

### High Priority (Semantic Alignment)

1. **Warning Color**: Change `colour226` ‚Üí `colour220`
   - Current: `colour226` (#ffff00) - Pure yellow
   - Recommended: `colour220` (#ffd700) - Closer to `--accent-yellow: #f59e0b`
   - Impact: Better matches dashboard warning indicators

2. **Success Color**: Add `colour82` for health/status
   - New: `colour82` (#5fff87) - Matches `--accent-green: #10b981`
   - Usage: Health checks, success indicators
   - Impact: Consistent success indication across terminal/dashboard

### Medium Priority (Module Colors)

3. **Module-Specific Colors**: Add department colors
   - Core: `colour51` (#00ffff) - Matches `--dept-api: #00d4ff`
   - Analytics: `colour99` (#875fff) - Matches `--dept-dashboard: #667eea`
   - Research: `colour93` (#d75fff) - Matches `--dept-orca: #9c27b0`
   - Monitoring: `colour166` (#d75f00) - Already matches `--dept-security`

4. **Cyan Alignment**: Consider `colour51` vs `colour33`
   - Current: `colour33` (#00afff) - Slightly darker cyan
   - Alternative: `colour51` (#00ffff) - Closer to `--accent-cyan: #00d4ff`
   - Trade-off: `colour33` is more readable, `colour51` is more accurate

### Low Priority (Polish)

5. **Text Colors**: Fine-tune for better readability
   - Primary text: `colour254` (#e4e4e4) - Closer to `--text-primary`
   - Secondary text: `colour245` (#8a8a8a) - Matches `--text-secondary`

## üîß Quick Implementation

### Update Warning Color

```tmux
# In config/.tmux.conf, line 134
# Change:
fg=colour226  # Old: Pure yellow
# To:
fg=colour220  # New: Gold/yellow (closer to #f59e0b)
```

### Add Success Color

```tmux
# Add to status-right for health indicators:
#[fg=colour82]OK:#{health_count}
```

### Module-Specific Status Colors

```bash
# In scripts/tmux-mlgs.sh, add after session creation:
case "$MODULE" in
    core)
        tmux set-option -t "$SESSION_NAME" status-style "bg=colour234,fg=colour51"
        ;;
    analytics)
        tmux set-option -t "$SESSION_NAME" status-style "bg=colour234,fg=colour99"
        ;;
    research)
        tmux set-option -t "$SESSION_NAME" status-style "bg=colour234,fg=colour93"
        ;;
    monitoring)
        tmux set-option -t "$SESSION_NAME" status-style "bg=colour234,fg=colour166"
        ;;
esac
```
