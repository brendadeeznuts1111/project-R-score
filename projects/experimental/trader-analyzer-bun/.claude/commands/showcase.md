# NEXUS Showcase UI

Interactive browser UI for API exploration.

## [instructions]

Access the Showcase UI for testing NEXUS endpoints.

```bash
cd /Users/nolarose/Projects/trader-analyzer-bun
```

## [access]

```bash
# Start server
bun run dev

# Open in browser
open http://localhost:3001/showcase
```

## [features]

| Feature | Description |
|---------|-------------|
| Live API Testing | Click-to-execute endpoints |
| Response Viewer | Formatted JSON output |
| Status Indicators | Real-time health status |
| Dark Theme | Eye-friendly interface |

## [endpoints.in.showcase]

- `/api/health` - System health
- `/api/orca/stats` - ORCA statistics
- `/api/orca/sports` - Supported sports
- `/api/arbitrage/status` - Scanner status
- `/api/deribit/index/btc` - BTC price
- `/api/cache/stats` - Cache metrics

## [code.location]

- Showcase Handler: `src/api/showcase.ts`
- Routes: `src/api/routes.ts`

## [customization]

The showcase is a single HTML response generated by Hono. Modify `src/api/showcase.ts` to add new endpoints or change styling.
