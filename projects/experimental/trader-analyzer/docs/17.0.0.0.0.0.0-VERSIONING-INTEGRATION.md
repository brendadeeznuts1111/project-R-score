## üìã Version Format

**Design System versions follow the pattern**: `17.X.Y.Z.A.B.C`

Where:
- `17` - Major subsystem (Design System & Theming)
- `X` - Major component (0 = Core System)
- `Y` - Component category (0 = Core, 1 = Integration, 2 = Browser)
- `Z` - Feature group
- `A` - Sub-feature
- `B` - Implementation detail
- `C` - Specific variant
