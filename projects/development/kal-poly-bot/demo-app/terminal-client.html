Getting Started

  1. Start the Bun PTY server: bun src/terminal-server.ts
  2. Configure server URL and shell type
  3. Click "Connect" to create a new session
  4. Type commands directly in the terminal

Features

  • Real-time bidirectional communication
  • Support for TTY-aware applications (vim, htop)
  • Terminal resize support
  • Session management and auto-cleanup
  • Security validation and input sanitization
