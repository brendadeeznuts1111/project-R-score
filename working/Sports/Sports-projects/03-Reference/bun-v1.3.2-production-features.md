**Reliable cleanup** for database connections and servers:

```typescript

});
```

**See**: [[Testing Best Practices]]

### 3. WebSocket Subscription Management

**`ws.subscriptions` getter** for debugging and management:

```typescript
websocket: {
  open(ws) {
    console.log('Active subscriptions:', ws.subscriptions);
    ws.subscribe("heartbeat");
  },
  
  close(ws) {
    ws.subscriptions.forEach(topic => {
      this.cleanupTopicResources(topic, ws);
    });
  }
}
```

**See**: [[WebSocket Services]]
