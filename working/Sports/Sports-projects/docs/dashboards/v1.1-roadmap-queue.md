## Unified Phases Queue

```dataview
TABLE file.link AS "Suggestion", suggestionType, confidence, priority, roadmapPhase, biasScore
FROM [[forge]] AND [[suggestion]] AND [[v1]].1
WHERE confidence >= 0.75
FLATTEN responseActions AS actions
SORT dueDate ASC
GROUP BY roadmapPhase
```

## JavaScript Variant (Advanced Metrics)

```dataviewjs
const suggestions = dv.pages("#forge AND [[v1]].1").where(p => p.confidence >= 0.8 && p.roadmapPhase);
const phaseCounts = suggestions.groupBy(p => p.roadmapPhase);
const totalROI = suggestions.reduce((sum, p) => sum + (p.impact || 0.08), 0) / suggestions.length;

dv.header(3, `v1.1 Unified Queue: ${suggestions.length} Items | Avg ROI: ${totalROI.toFixed(2)} | Phases Active: ${Object.keys(phaseCounts).join(', ')}`);
dv.table(["Phase", "Count"], Object.entries(phaseCounts).map(([k, v]) => [k, v.length]));

// Gov Alert: Low bias triggers review
if (suggestions.some(p => parseFloat(p.biasScore) < 0.9)) {
  dv.paragraph("⚠️ Bias Alert - Ethical Review Needed (Phase 6 Focus)");
}
```
