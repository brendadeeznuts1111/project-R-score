# Archived Documents

This directory contains historical documentation and status files from the project development.

## Contents

### Status/Completion Files
These files tracked the completion status of various features during development:
- `*_COMPLETE.md` - Feature completion trackers
- `ANSI_WRAPPING_ADDED.md` - Documentation status
- `JSDOC_DOCUMENTATION.md` - Documentation tracking
- `LINT_*.md` - Linting status files

### Profiling Integration Files
Historical documentation for Bun v1.3.7 profiling integration:
- `BUN_v1_3_7_*.md` - Version upgrade and feature validation docs
- `CPU_PROFILING_*.md` - CPU profiling guides and completion status
- `HEAP_PROFILING_*.md` - Heap profiling documentation
- `ENHANCED_*_COMPLETE.md` - Enhanced profiling feature tracking

### Commit Organization
- `PROFESSIONAL_*_GUIDE.md` - Git workflow and commit organization guides
- `ENHANCED_COMMIT_*.md` - Commit structure documentation

## Note

For current documentation, see the main `docs/` directory.
