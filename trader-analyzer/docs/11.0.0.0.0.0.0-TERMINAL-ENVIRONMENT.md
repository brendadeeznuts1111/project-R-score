## 11.5.0.0.0.0.0: Troubleshooting

### 11.5.1.0.0.0.0: Common Issues & Solutions

#### 11.5.1.1.0.0.0.0: Tmux Not Starting

```bash
# Check tmux version
tmux -V  # Should be >= 3.0

# If not installed:
# macOS
brew install tmux

# Ubuntu/Debian
sudo apt install tmux

# Arch Linux
sudo pacman -S tmux
```

#### 11.5.1.2.0.0.0.0: Colors Not Working

```bash
# Ensure terminal supports 256 colors
echo $TERM  # Should be xterm-256color or similar

# Set in your shell profile (~/.bashrc or ~/.zshrc)
export TERM=xterm-256color
```

#### 11.5.1.3.0.0.0.0: Bun Console Errors

```bash
# Check Bun version
bun --version  # Should be >= 1.0

# Install dependencies
bun install

# Clear console cache (if using history file)
rm -f .bun-console-history
```

#### 11.5.1.4.0.0.0.0: Plugin Issues

```bash
# Install TPM if missing
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

# Install plugins
# 1. Start tmux
# 2. Press Ctrl-Space + I (capital i)
```

#### 11.5.1.5.0.0.0.0: Performance Issues

```bash
# Reduce history limit in ~/.tmux.conf
set -g history-limit 50000  # Instead of 100000

# Disable unused plugins
# Comment out in ~/.tmux.conf:
# set -g @plugin 'tmux-plugins/tmux-continuum'
```

## 11.14.0.0.0.0.0: Integrated Contextual Discovery via Ripgrep

### 11.14.1.0.0.0.0: Cross-Language & Cross-Tool Discovery Capabilities

The consistent version numbering scheme (`11.x.x.x.x.x.x`) acts as a universal identifier that transcends file types (TypeScript, Markdown, YAML, Shell, Tmux config) and tool domains (code, configuration, scripts, API calls, terminal environment).

#### 11.14.1.1.0.0.0.0: Unified Search Schema

The version numbering enables holistic search across:
- **TypeScript**: Implementation code (`scripts/bun-console.ts`)
- **Shell Scripts**: Launch and setup scripts (`scripts/tmux-mlgs.sh`, `scripts/setup-terminal.sh`)
- **Markdown**: Documentation (`docs/11.0.0.0.0.0.0-TERMINAL-ENVIRONMENT.md`)
- **Configuration**: Tmux config (`config/.tmux.conf`), Tmuxinator (`config/.tmuxinator.yml`)
- **Tests**: Test suites (`test/bun-console.test.ts`)
- **Package.json**: NPM scripts and metadata
