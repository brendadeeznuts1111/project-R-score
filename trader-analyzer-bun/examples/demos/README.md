# Demo Scripts

This directory contains demonstration scripts showcasing various Bun APIs and features used in the NEXUS Trading Platform.

## Purpose

These scripts serve as:
- **Learning resources** - Examples of Bun-native APIs and patterns
- **Testing utilities** - Tools for testing and debugging features
- **Reference implementations** - Code samples for common use cases

## Scripts Overview

### HTML Rewriter Demos
- `demo-html-rewriter.ts` - Basic HTML rewriter example
- `demo-html-rewriter-simple.ts` - Simplified HTML rewriter
- `demo-html-rewriter-comparison.ts` - Comparison of different approaches
- `demo-html-rewriter-live-editor.ts` - Interactive HTML editor
- `demo-html-rewriter-server.ts` - Server-side HTML rewriting

### Bun Utilities
- `demo-bun-utils.ts` - Bun utility functions showcase
- `demo-bun-spawn-complete.ts` - Complete Bun.spawn() examples
- `demo-bun-inspect-custom.ts` - Custom Bun.inspect() examples

### Data Structures
- `demo-circular-buffer.ts` - Circular buffer implementation demo
- `demo-advanced-circular-buffer.ts` - Advanced circular buffer features

### Debugging & Testing
- `demo-fetch-debug.ts` - Fetch debugging utilities
- `demo-console-features.ts` - Console enhancement features
- `demo-worker-threads.ts` - Worker threads examples

### Tag Management
- `demo-tag-manager-pro.ts` - Advanced tag manager demo
- `tag-manager.ts` - Basic tag manager
- `tag-manager-pro.ts` - Professional tag manager

### Utilities
- `fix-type-errors.ts` - TypeScript error fixing utility

## Usage

Run any demo script with Bun:

```bash
bun examples/demos/demo-html-rewriter.ts
```
